# AbhiBus Website Automation Testing (Selenium)

## Description
This project is an automated testing application developed using Selenium WebDriver to test the functionality of the AbhiBus website. It validates core features such as login, bus search, seat selection, booking flow, and page navigation to ensure the website works correctly.

## Features
- Automated browser testing using Selenium WebDriver
- Bus search validation
- Source and destination selection testing
- Date picker testing
- Seat selection testing
- Booking workflow testing
- UI element validation

## Technologies Used
- Java
- Selenium WebDriver
- Eclipse IDE

## Project Structure
- src/test/java →come.abhibus.test
- src/main/java →come.abhibus.pageobject
- drivers/ → Browser drivers


## Test Scenarios Covered
1. Verify homepage loads successfully
2. Verify user can search buses
3. Verify source and destination fields
4. Verify calendar/date selection
5. Verify available buses are displayed
6. Verify seat selection process
7. Verify booking page navigation
8. Verify invalid input handling

## How to Run the Project
1. Clone/download this repository
2. Open in Eclipse IDE
3. Install dependencies 
4. Update browser driver path
5. Run test classes
